# Production System - Workflow (Draft)

## Steps
1. Select Recipe.
2. Select # batches.
3. Populate Recipe chooser (INGREDIENT, UOM, QUANTITY).
4. Select an INGREDIENT row and open filtered item search.
5. Pick ITEM and enter QUANTITY (writes to Inventory item chooser).
6. Check_invSys updates for all selected rows.
7. To USED (stage components).
8. Confirm MADE (move finished qty to MADE).
9. Send to TOTAL INV (clear staging).

## Note
- Batch / recall codes are captured before Confirm MADE.
